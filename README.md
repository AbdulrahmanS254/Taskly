# 📌 Project Management System (Frontend)

This is the modern Frontend client for the **Project Management System**.
The application is designed to help teams organize, track, and collaborate on work using a structured workflow: **Projects → Epics → Tasks**.

---

## 🛠️ Tech Stack & Tools (Current Setup)

*   **Framework:** React (Vite)
*   **Language:** TypeScript
*   **Package Manager:** pnpm
*   **Code Quality:** ESLint + Prettier

---

## 🧩 Key Features (Under Development)

*   🔐 **Authentication:** Secure signup, login, and token-based project invitation system.
*   📁 **Project & Epics Management:** Create, edit, and track projects and their sub-epics.
*   ✅ **Task Management:** Full workflow from `TO_DO` to `DONE`.
*   📊 **Board View:** Interactive Kanban board with drag-and-drop capability.
